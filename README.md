# ChatNow

### User Story

```
As a User
I want to make a social chat room where you can chat one on one and in a group chat.

```

---

### Acceptance Criteria

```md
When you enter the app
You are able to put your login information
If you don't have an account
You can register
Once you are logged in
you can see who is in the chat room. You can search and chat with anyone one-on-one.
Or you can form a group chat. When you click on profile, you are able to see their profile
information. You can log out after your session.

```
